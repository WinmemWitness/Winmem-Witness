# Winmem Packages

This folder contains reusable libraries and developer tooling used by Winmem apps.

- cli: self-hosting CLI (`winmem`)
- sdk-js: JavaScript/TypeScript client SDK
- core: domain models + engines + policies
- solana: Solana RPC pool + parsers + indexing utilities
- storage: Prisma schema + repositories + blob/cache/locks
- cryptography: canonical hashing + Merkle tree + optional signatures
- llm: providers + prompts + guardrails (optional)
- observability: logger/metrics/tracing/health/redaction
- ui-kit: retro UI primitives shared by web/docs-site

All content is English-only.
